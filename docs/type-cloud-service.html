Onboarding of Cloud Services

Cloud Service is a specialized product type that extends syndicated products functionalities by adding a cloud credential requirement.

In this scenario, the customers must have a cloud credential already associated with their account, compatible with the cloud service they want to acquire.

As an example, the Kubernetes clusters is an implementation of a Cloud Service.
