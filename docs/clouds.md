| Name            | Linux Stacks | .NET (Windows) | Notes                              |
|-----------------|--------------|----------------|------------------------------------|
| Amazon AWS      | V            |                | VPC instances not yet supported    |
| Digital Ocean   | V            |                | No additional data disks available |
| Enter the Cloud | V            |                |                                    |
| Google Compute  | V            |                |                                    |
| IBM Softlayer   | V            |                |                                    |
| Joyent          | V            |                |                                    |
| Microsoft Azure | V            |                |                                    |
| Rackspace       | V            | V              | Data disk minimun size 100 GB      |
